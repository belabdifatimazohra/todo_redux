## Installation

This App requires [Node.js](https://nodejs.org/) v10+ to run.

Install the dependencies and devDependencies and start the server.

```sh
cd todo_redux
npm i
npm start 
```
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
